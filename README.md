# 🎨 Figma Shortcut

Website sederhana yang berisi kumpulan panduan *keyboard shortcuts* penting di Figma. Project ini dibuat untuk mempermudah para desainer UI/UX dalam mempercepat *workflow* kerja mereka tanpa harus menghafal semuanya satu per satu.

🚀 **[Lihat Demo Website](https://figmashortcut.netlify.app/)**

---

## ✨ Fitur Utama
* 📋 **Kategori Terstruktur**: Shortcut dikelompokkan dengan rapi berdasarkan fungsinya (Tools, View, Arrange, dll).
* 📱 **Responsive Layout**: Desain yang fleksibel dan nyaman dibaca, baik lewat layar laptop maupun HP.
* ⚡ **Ringan & Cepat**: Menggunakan struktur kode yang efisien sehingga *loading* website terasa instan.

## 🛠️ Tech Stack
* **HTML5** - Struktur elemen halaman web.
* **CSS3** - *Styling*, pewarnaan, dan tata letak (*Layouting*).
